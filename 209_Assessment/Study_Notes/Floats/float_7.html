I have been using a clear fix technique on all of these example without saying what it does. When all the child elements of a parent are floated they are all removed from the normal flow of the page. This means that there is nothing inside the parent to keep its dimensions which cause borders to collapse, among other things. This is demonstrated here with the red border of the main element.
