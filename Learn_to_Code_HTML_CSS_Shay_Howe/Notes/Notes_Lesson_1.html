Building Your First Web Page

Setting Up the HTML Document Structure

Below is the breakdown of a typical HTML document structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a web page.</p>
  </body>
</html>

Self-Closing Elements

Some elements are assigned content with the use of the charset attribut and value. Some common self-closing element include: <br> <img> <meta> <wbr> <embed> <input> <param> <hr> <link> <source> .

Code Validation

The W3C has abuilt HTML and CSS validators that will scan code for mistakes.
