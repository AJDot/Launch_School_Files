Creating Lists

There are 3 types of lists: unordered, ordered, and description. The choice between them is made semantically.

Unordered

Order obviously doesn't matter. Nothing here that you don't already know.

Ordered Lists

Order does matter here. Ordered lists also have unique attributes available to them including start and reversed.

Start Attribute

Defines the number from which an ordered list should start. Default is 1. The start attribute is used on the <ol> element.

<ol start="30">
  <li>Head north on N Halsted St</li>
  <li>Turn right on W Diversey Pkwy</li>
  <li>Turn left on N Orchard St</li>
</ol>

Reversed Attribute

Does what you think, counts down instead of up. The attribute doesn't accept any value. It is ether true or false where false is the default value. Just type "reversed" into the ordered list tag to make it so.

<ol reversed>
  <li>Head north on N Halsted St</li>
  <li>Turn right on W Diversey Pkwy</li>
  <li>Turn left on N Orchard St</li>
</ol>

Value Attribute

This can be used on any <li> element within an ordered list to change its value.

<ol>
  <li>Head north on N Halsted St</li>
  <li value="9">Turn right on W Diversey Pkwy</li>
  <li>Turn left on N Orchard St</li>
</ol>

Description Lists

Used to outline multiple terms and their descriptions, like a glossary. Use the <dl> element then instead of using <li> elements inside it, use the <dt> elements for each term and the <dd> for each description.

A description can have multiple terms. A term can have multiple descriptions. A <dt> element, however, must always come before a <dd> element.

Description Lists Demo

<dl>
  <dt>study</dt>
  <dd>The devotion of time and attention to acquiring knowledge on an academic subject, especially by means of books</dd>
  <dt>design</dt>
  <dd>A plan or drawing produced to show the look and function or workings of a building, garment, or other object before it is built or made</dd>
  <dd>Purpose, planning, or intention that exists or is thought to exist behind an action, fact, or material object</dd>
  <dt>business</dt>
  <dt>work</dt>
  <dd>A person's regular occupation, profession, or trade</dd>
</dl>

List Style Properties

Know that these exist:

  • list-style-type (can even use an image)
  • list-style-position

Horizontally Displaying List

there are a few ways to display lists as a single line:

  • making the display property value of <li> element inline or inline-block
  • floating the <li> elements

Note: Changing to inline or inline-block will remove the item marker.

Floating List

If the item marked is needed, floating each list item is necessary. Will need horizontal margin and padding to make the item markers stop displaying on top of the previous list item.
