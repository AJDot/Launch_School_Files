Working with Typography

Web typography is now a complex subject. We needed the development of a system for embedding our own web fonts on a website.

Typeface vs. Font

Often they are interchangeable words.

Typeface: what we see - the look, feel, readability of type.

Font: a file that contains a typeface.

Adding Color to Text

One of the first decisions to make when building a website is deciding on the primary typeface and text color. Color of text is easily changed:

html {
  color: #555;
}

Changing Font Properties

There are two categories of properties to do so: font-based and text-based. These are accessed using font-* and text-*

Font Family

Declare the font and the fallback font in case the primary font does not work using the font-family property.

Font Size

Self-explanatory

Font Style

Values: normal, italic, oblique, and inherit

Font Variant

Values: normal, small-caps, and inherit

Font Weight

Values: normal, bold, bolder, lighter, and inherit

For bold, can use 100, 200, 300, 400, 500, 600, 700, 800, and 900 where 400 is normal and 700 is bold. Can only be used for fonts that provide the typeface in the desired weight.

Line Height

This is the distance between two lines of text and is set using the line-height property.

The best practice is the set the line height to ~1.5 times the font-size property value. This can be done by setting the line-heighth to 150%, or just 1.5.

Line height may also be used to vertically center text within an element by setting the line-height and height properties to the same value.

Applying Text Properties

These properties assist to align, decorate, indent, transform, and space text.

Text Align

Values: left, right, center, justify, and inherit

Text Decoration

Values: none, underline, overline, line-through, and inherit

Text Indent

Used to indent the first line of text within an element. Negative values will outdent, positive values will indent.

Text Shadow

Adds a shadow (or multiple shadows) to text. Takes usually 4 values - 3 lengths and a color - horizontal offset, vertical offset, blur radius, color.

/* The text-shadow property here is casting a 30% opaque black shadow 3 pixels towards the right, 6 pixels down, and blurred 2 pixels off all <p> element text: */

p {
  text-shadow: 3px 6px 2px rgba(0, 0, 0, .3);
}

Text Transform

Values: none, capitalize, uppercase, lowercase, and inherit

Letter Spacing

The larger the value, the farther apart the letters will be from each other

Word Spacing

I think you can guess what this does.

Using Web-Safe Fonts

There are only a few fonts pre-installed on every computer, table, etc. These are the only truly safe fonts to use freely since they will always render properly.

  • Arial
  • Courier New, Courier
  • Garamond
  • Georgia
  • Lucdia Sans, Lucida Grande, Lucida
  • Palatino Linotype
  • Tahoma
  • Times New Roman, Times
  • Trebuchet
  • Verdana

Embedding Web Fonts

We can upload fonts to a server and include them on a website using @font-face.

In the CSS, identify the font's name using the font-family property and then the source using the src property.

@font-face {
  font-family: "Lobster";
  src: local("Lobster"), url("lobster.woff") format("woff");
}
body {
  font-family: "Lobster", "Comic Sans", cursive;
}

Warning

The ability to embed any typeface on a website does not mean we are legally able to do so. We must abide by the licensing we have been warranted.

However, some companies use subscriptions (e.g. Typekit) to deal out fonts and others are completely free (e.g. Google Fonts).

Including Citations & Quotes

  • <cite>: Used to reference a creative work, author, or resource
  • <q>: Used for short, inline quotations
  • <blockquote>: Used for longer external quotations

Citing a Creative Work

The element must include either the title of the work, the author's name, or a URL reference to the work. By default it will appear in italics.

Dialogue & Prose Quotation

Often quoted inline. Used to indicate the prose shouldn't be used for any other purpose.

Dialogue & Prose Citation

Optionally you can include a cite attribute on the q element. This will add value to screen readers but does not show on the page.

<p><a href="http://www.businessweek.com/magazine/content/06_06/b3970001.htm">Steve Jobs</a> once said, <q cite="http://www.businessweek.com/magazine/content/06_06/b3970001.htm">One home run is much better than two doubles.</q></p>

External Quotation

Large text that comes from an external source. It can even have headings and paragraphs included inside it.

<blockquote>
  <p>“In most people’s vocabularies, design is a veneer. It’s interior decorating. It’s the fabric of the curtains, of the sofa. But to me, nothing could be further from the meaning of design. Design is the fundamental soul of a human-made creation that ends up expressing itself in successive outer layers of the product.”</p>
</blockquote>
