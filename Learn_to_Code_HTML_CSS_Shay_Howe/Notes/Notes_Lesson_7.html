Setting Backgrounds & Gradients

Backgrounds have more of an impact than you think on web design. Backgrounds can:

  • create look and feel
  • establish groupings
  • assign priority
  • influence usability

Element backgrounds can be a solid color, an image, a gradients or a combination of these. The goal is to place everything with a purpose. If you don't know why you are adding that background or that image, then don't.

Adding a Background Color

Use the background or background-color property. Background is for colors and images while background-color is only for colors.

We can use keywords, hexadecimal codes, and RGB, RGBa, HSL, and HSLa values.

Transparent Backgrounds

When using these it's a good idea to provide a fallback coolor because not all browsers recognize RGBa and HSLa values. Since CSS is cascading, add the fallback color on top of the more advanced RGBa/HSLa color. The advanced color will be ignored if it is not recognized by the browser.

Adding a Background Image

Must include the image source url using "url()". This can reference local paths as well as absolute. Using just this method is typically undesirable. Need to use background-repeat and background-position to control the image.

Shorthand Background Image Values

div {
  background: #b2b2b2 url("alert.png") 20px 10px no-repeat;
}

Designing Gradient Backgrounds

Introduced in CSS3, they are not backwards compatible. Gradient backgrounds are treated as background images. We create them using the background or background-image properties just like a regular background image. Vendor prefixes are still needed for this.

Linear Gradient Background

Linear Gradient Example

div {
  background: #466368;
  background: -webkit-linear-gradient(#648880, #293f50);
  background:    -moz-linear-gradient(#648880, #293f50);
  background:         linear-gradient(#648880, #293f50);
}

Changing the Direction of a Gradient Background

div {
  background: #466368;
  background: linear-gradient(to right bottom, #648880, #293f50);
}

If an element isn't exactly square, the background gradient will not proceed directly from one corner to the other. It uses something called "magic corners." A description of what this is can be found here, "Linear Gradient Keywords", by Eric Meyer.

Beside keywords, degress can be used. Ex: 135deg will move to teh right bottom of an element.

Radial Gradient Background

They work just like linear gradients just with a different property.

Radial Gradient Background Demo

div {
  background: #466368;
  background: radial-gradient(#648880, #293f50);
}

Radial gradients are more complex. You can change values for location, size, radius and others. Read radial gradients for more info. There are even CSS3 gradient generators you can use to create these gradients.

Gradient Color Stops

We can add multiple colors to transition between. To do this we add color stops to the given gradients function, with commas separating each color stop from the next.

Gradient Color Stops Demo

div {
  background: #648880;
  background: linear-gradient(to right, #f6f1d3, #648880, #293f50);
}
